# Task Runners

Utilities for running tasks from the command line.

## Invoke

Pythonic task management & command execution.
Invoke is a Python library for managing shell-oriented subprocesses and organizing executable Python code into CLI-invokable tasks.

- https://www.pyinvoke.org/
- https://github.com/pyinvoke/invoke

## Just

just is a handy way to save and run project-specific commands.

- https://github.com/casey/just

## Poe the Poet

Poe the Poet is a batteries included task runner that works well with poetry or with uv.

- https://poethepoet.natn.io/
- https://github.com/nat-n/poethepoet

## Task

A task runner / simpler Make alternative written in Go.

- https://taskfile.dev/
- https://github.com/go-task/task
